Projects

CS61B Database

A relational database management system that I created with Java that can store multiple types of data in tables. I also built a domain specific language similar to the SQL language to interact with the database.

BearMaps

A map of the UC Berkeley campus that I created with Java using a QuadTree data structure. I also implemented A* in order to find the shortest path between two locations on the map.

SIXT33N Car

A miniature car that can obey certain voice commands. I constructed a front-end circuit accompanied with a mic circuit board that converts sound to a controlled signal. I also programmed an MSP430 Launchpad with Arduino code to perform principal component analysis to classify voice commands and to control the actions of the car.

Loop Pedal

A project I worked on during CalHacks 3.0 where my group and I created an iOS application that records sound files and combines them in a loop. All recording is controlled by a bluetooth-linked Arduino by a switch on a circuit. You can view the project here.

Bear's Sunlight Clock

A project I worked on during Iot Hacks where my group and I created an alarm clock with LEDs to imitate sunlight. We programmed an Arduino with C++ so that it activates the LEDs at a specified time and linked the Arduino to a website through WiFi module to get the time. You can view the project here.
